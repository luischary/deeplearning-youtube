Bem-vindo! Este repositório contém todo o código fonte desenvolvido nos vídeos do canal. O foco aqui é **Deep Learning**, **Implementação de Papers** e **Matemática de Machine Learning**, quase sempre utilizando **PyTorch** ou Python puro para construir algoritmos do zero.

O objetivo é preencher a lacuna entre a teoria acadêmica e a prática de engenharia. Aqui você não vai apenas aprender a usar uma biblioteca, mas entender o que acontece "debaixo do capô" dos modelos de IA modernos.

## 🚀 Como usar este repositório

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/luischary/deeplearning-youtube.git
    cd deeplearning-youtube
    ```

2.  **Crie um ambiente virtual (Recomendado):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # No Windows: venv\Scripts\activate
    ```

3.  **Instale as dependências:**
    A maioria dos projetos utiliza as bibliotecas listadas no `requirements.txt`.
    ```bash
    pip install -r requirements.txt
    ```
